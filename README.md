Components:
- Raspberry Pi Pico 2W (any Pico series will work - pre-soldered headers)
https://www.adafruit.com/product/6315
- microUSB cable (make sure other end matches your PC/laptop, USB-A or USB-C. Also MUST be a data cable
https://www.adafruit.com/product/2185
- Breadboard - Monk Makes has Pico pin info silk screened on the board
https://www.adafruit.com/product/5422
- 8 pin-pin jumper wires
https://www.adafruit.com/product/1957
- NeoPixel light strip
https://www.adafruit.com/product/3919
- Speaker
https://www.adafruit.com/product/3351d
- AMP
https://www.adafruit.com/product/5647
- Potentiometer
https://www.adafruit.com/product/4493
- two 3 pin JST connector (one for app, one for potentiometer
https://www.adafruit.com/product/3893
- 180° Servo
https://www.adafruit.com/product/169

When wired & plugged in, the code.py file above should light up the NeoPixel lights (it assumes 150 lights, so be aware this will look differently if you have different lights), it will play the "wondering.wav" sound in the "sounds" folder, and if you rotate the potentiometer, you should see the servo motor rotate, as well.
